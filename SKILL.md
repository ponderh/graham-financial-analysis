# 格雷厄姆式A股财务报表分析（增强版 v2）

## 工作流程（优化后）

### 阶段1：交易所财报PDF获取（scripts/exchange_report_fetcher.py）

**直接从上交所/深交所API获取真实财务数据PDF（推荐优先使用）：**

```bash
python3 scripts/exchange_report_fetcher.py <股票代码> <输出目录> [--type annual|half|quarterly|all] [--max N]
```

| 报告类型 | 说明 |
|---------|------|
| `--type annual` | 仅年报 |
| `--type half` | 仅半年报 |
| `--type quarterly` | 仅季报 |
| `--type all` | 全部财报（默认） |
| `--max 3` | 最多下载3份（默认5） |

**功能：**
- 对接上交所/深交所官方API（真实数据源）
- 自动识别科创板(688)/深交所股票
- 下载年报+半年报+季报PDF
- **用pdfplumber解析PDF，提取：**
  - 财务报表（利润表/资产负债表/现金流量表完整数据）
  - **财务报表附注（最重要！）**：应收账款账龄结构、关联交易明细、担保/或有事项、存货分类、分部信息
  - 分季度数据（Q1/Q2/Q3/Q4单季度数据）
  - 董事会报告经营分析
  - **关键风险警示检测**（审计保留意见、持续经营存疑、会计政策变更、商誉减值、3年以上应收账款等）

### 阶段2：akshare数据采集（scripts/data_fetch.py）

**作为PDF数据的补充，采集结构化指标：**

```bash
python3 scripts/data_fetch.py <股票代码> <输出目录>
```

- 综合财务指标（akshare东方财富，86+字段）
- 利润表、资产负债表、现金流量表（3年）
- 公司基本信息
- 股价历史（前复权）

### 阶段3：年报PDF下载备选（scripts/pdf_download.py）

**当exchange_report_fetcher.py失败时使用此模块：**

```bash
python3 scripts/pdf_download.py <股票代码> <输出目录>
```

### 阶段4：多估值体系计算（scripts/valuation_multi.py）

**6种估值方法，必须全部计算：**

```python
python3 scripts/valuation_multi.py <股票代码> <数据目录>
```

| 方法 | 分类 | 说明 |
|------|------|------|
| ① Graham Number | 格雷厄姆 | 保守版 + 成长调整版 |
| ② Earnings Power Value (EPV) | 格雷厄姆改良 | 基于可持续EPS/国债收益率 |
| ③ Buffett Intrinsic Value | 巴菲特 | Owner Earnings × 10年国债收益率倒数 |
| ④ Two-Stage DCF | 主流DCF | 高速增长期(5年) + 永续折现 |
| ⑤ PEG Ratio | 相对估值 | PE/未来3年增速 |
| ⑥ Dividend Discount Model | 红利折现 | 适用于高分红股 |
| ⑦ EV/EBITDA | 相对估值 | 剔除财务杠杆影响 |

## 各估值方法详解

### ① Graham Number（格雷厄姆数）
```
保守版：V = √(22.5 × EPS × BPS)
成长调整版（ROE>15%）：V = √(22.5 × EPS × BPS × (1 + 0.5×(ROE-rf)))
```

### ② Earnings Power Value (EPV)
```
V = EPS / r
其中 r = 股权要求收益率 = 10年国债收益率 + 股债溢价(3%)
    = 3% + 3% = 6%（A股适用）
V_adj = EPV × (净现比修正因子)
净现比修正因子 = min(净现比/1.0, 1.0)
```

### ③ Buffett Intrinsic Value
```
内在价值 = Σ (OE_t / (1+r)^t) + TV / (1+r)^n
OE = 净利润 + 折旧摊销 - 维持性资本支出
简化版（永续增长假设）：
V = OE × (1+g) / (WACC - g)
WACC = 8%（A股股权成本）
g = 永续增长率（取无风险利率3%上限）
```

### ④ Two-Stage DCF
```
第一阶段（高增长期，5年）：
OE_1 = FCF估算值（净利润 + D&A - 维持性capex）
CF_1 = OE_1 × (1+增速)^t，贴现到现值

第二阶段（永续期）：
TV = CF_5 × (1+g) / (WACC - g)
TV_discounted = TV / (1+WACC)^5

总价值 = Σ第一阶段CF + TV_discounted
```

### ⑤ PEG Ratio
```
PEG = PE / (净利润增速×100)
PEG < 1：相对低估；PEG > 1.5：相对高估
目标PE = 增速(小数) × G因子（G因子=15为格雷厄姆上限）
```

### ⑥ Dividend Discount Model
```
V = D / (r - g)
D = 已知分红（假设稳定）
r = 股权要求收益率（8%）
g = 永续增长率（3%）
仅适用于分红稳定的高分红股（分红率>30%）
```

### ⑦ EV/EBITDA
```
EV = EBITDA × EV/EBITDA倍数
倍数来源：行业平均倍数 × (1 ± 折扣溢价)
EBITDA = 净利润 + 利息支出 + 税 + 折旧摊销
```

## 关键阈值速查（更新版）

| 指标 | 安全 | 警示 | 危险 |
|------|------|------|------|
| 净现比 | >1.0 | 0.8-1.0 | <0.8 |
| 资产负债率（制造业） | <40% | 40-50% | >50% |
| ROE | >15% | 10-15% | <10% |
| 格雷厄姆数折扣 | >20%折扣 | 10-20% | <10% |
| 有息负债率 | <20% | 20-40% | >40% |
| DCF股价 vs 当前价 | <0.85 | 0.85-1.0 | >1.0 |

## 同业比较行业分类（akshare来源）

使用 `ak.stock_board_industry_name_em()` 和 `ak.stock_board_industry_cons_em()` 获取行业成分股，再选取申万二级行业分类作为peer group基准。

**选取peer标准：**
- 同一行业（申万二级）
- 市值在0.5x~2x范围内
- 上市满3年（剔除次新股）
- 选取5-10家可比公司
