# SKILL_DEEP.md — 深度年报解析框架 v2.0

> 本框架是格雷厄姆分析Skill的深度增强模块，解决"年报PDF海量信息如何充分利用"的问题。

---

## 核心设计

### 四文档体系

| 文档类型 | 频率 | 解析深度 | 时间窗口 |
|---------|------|---------|---------|
| **年报PDF** | 每年1次 | 8章节全覆盖 | 4月发布 |
| **半年报PDF** | 每年2次 | 核心章节 | 8月+3月 |
| **季报PDF** | 每季度1次 | 关键指标 | 4/7/10月 |
| **公告PDF** | 不定期 | 事件驱动触发 | 随时 |

**架构原则：年报为主干，半年报为补充，季报为追踪，公告为触发。**

---

## 四专家框架（v2.0增强版）

### Graham专家 — 财务附注深度

**分析内容：**
- 收入确认政策（激进/保守/合理判断）
- 坏账计提（预期信用损失模型参数提取）
- 研发资本化时点（最关键！临床阶段vs批准阶段）
- 审计意见（关键审计事项）
- 非经常性损益占比
- 关联交易定价公允性

**核心逻辑：** 审计同时将"收入确认"和"研发资本化"列为关键审计事项 → 双重红旗 → Graham评分≤3

---

### Buffett专家 — 商业本质深度

**分析内容：**
- 管理层语调深度分析（乐观/中性/谨慎的语义判断）
- 护城河多维度验证（专利、临床数据、注册证等）
- 竞争格局文字分析（主要竞争对手识别）
- FCF生成能力评估
- 管理层行动vs言语一致性

**核心逻辑：** 语调过度乐观 → 可信度打折 → Buffett评分-2

---

### Marks专家 — 风险识别深度

**分析内容：**
- 已披露风险分类提取（行业/政策/经营/财务/法律）
- **第二层思维**：市场共识 vs 真实情况
- 隐藏风险推断（集采降价、本土失速、募投滞后等）
- 周期位置判断

**核心逻辑：** 寻找"公司说的"和"实际发生的"之间的差距

---

### Critic评估员 — 综合质疑深度

**分析内容（完整10项）：**
1. 股东集中度 + 解禁风险
2. 客户集中度
3. 供应商集中度
4. 募资使用进度
5. 商誉及减值
6. 高管薪酬与激励
7. 分红政策
8. 股权激励行权价
9. 在建工程
10. 异常关联交易

**核心逻辑：** 所有数据以"股东利益是否受损"为判断标准

---

## 新增：董事会报告追踪

**格式：**`年份-报告类型（如2025-年度报告）` → 核心要点

**主题分类：**
- 营收表现：收入增长/下滑数据
- 盈利能力：净利润/扣非净利/毛利分析
- 研发进展：新药/临床/注册进展
- 行业竞争：市场份额/竞争对手描述
- 战略规划：管理层对未来的表述
- 政策影响：集采/医保/合规
- 风险提示：公司自己承认的风险

**目的：** 将多期董事会报告串联分析，识别管理层表述的变化趋势

## 使用方法

### 命令行

```bash
# 完整分析（PDF + 四专家 + 格雷厄姆数据整合）
python3 scripts/parse_deep.py 688351 ./output \
    --experts all \
    --pdf-dir ./pdfs \
    --integrate-graham

# 仅Graham专家
python3 scripts/parse_deep.py 688351 ./output \
    --experts Graham \
    --pdf-dir ./pdfs

# 整合同股票格雷厄姆skill数据
python3 scripts/parse_deep.py 688351 ./output \
    --experts all \
    --pdf-dir ./pdfs \
    --data-dir ./graham_output \
    --integrate-graham
```

### 输出文件

```
{stock}_Graham_result.json    # Graham专家结果
{stock}_Buffett_result.json   # Buffett专家结果
{stock}_Marks_result.json     # Marks专家结果
{stock}_Critic_result.json   # Critic评估结果
{stock}_deep_v2_report.json # 综合报告（含董事会报告）
```

## parse_deep.py v2.0 关键优化

1. **一次性PDF扫描**：所有专家共享同一扫描结果，避免重复解析
2. **关键词预检**：整页快速预检，只对含关键词页面详细提取
3. **process_text=False**：pdfplumber加速约3倍
4. **语义语调分析**：超过关键词计数的简单判断，对完整文本做语义分析
5. **第二层思维**：Marks专家从"市场共识"和"隐藏真相"两个角度评估

## 数据源说明

- **PDF年报**：通过 `exchange_report_fetcher.py` 从深交所/上交所API下载
- **格雷厄姆skill数据**：akshare财务指标 + `valuation_multi.py` 估值结果
- **季报/公告**：从深交所/上交所/东方财富API获取
