# 格雷厄姆八维分析框架（详细版）

## 模块1：盈利质量分析

### 核心指标
| 指标 | 计算公式 | 阈值 | 数据来源 |
|------|---------|------|---------|
| ROE趋势 | 3年加权ROE均值+趋势线 | >15%且稳定 | 加权净资产收益率(%) |
| 非经常性损益占比 | (EPS - 扣非EPS)/EPS | <10%为佳 | 摊薄每股收益/扣非EPS |
| 净现比 | 每股经营CF/每股收益 | >0.8，趋势>绝对值 | 每股经营性现金流/每股收益 |
| 毛利率稳定性 | 3年毛利率CV=std/mean×100 | CV<15%为稳定 | 销售毛利率(%) |
| 三项费用率 | (销售+管理+财务费用)/营收 | 下降=改善 | 费用占营收比 |
| 扣非增速 | (今年扣非EPS-去年)/去年 | 与营收增速匹配 | 扣非净利润增速 |

### 盈利质量评分标准
- **9-10分**：ROE>20%，净现比>1.2，非经常占比<5%，毛利率CV<10%
- **7-8分**：ROE>15%，净现比>0.8，非经常占比<10%，毛利率稳定
- **5-6分**：ROE 10-15%，净现比0.6-0.8，有一定非经常损益
- **<5分**：ROE<10%或净现比<0.6或非经常占比>20%

## 模块2：财务稳定性分析

### 核心指标
| 指标 | 计算公式 | 格雷厄姆阈值 |
|------|---------|------------|
| 资产负债率 | 总负债/总资产 | <40%（制造业） |
| 流动比率 | 流动资产/流动负债 | >2（>1.5可接受） |
| 速动比率 | (流动-存货)/流动负债 | >1 |
| 现金比率 | 现金/流动负债 | >30% |
| 有息负债率 | (短期+长期借款)/总资产 | <20% |
| 利息保障倍数 | EBIT/利息支出 | >5倍 |

### 有息负债计算
```
有息负债 = 短期借款 + 长期借款 + 应付债券
（不含应付账款/应付票据等经营性负债）
```

## 模块3：运营效率分析

### 核心指标
| 指标 | 趋势判断 | 数据来源 |
|------|---------|---------|
| 总资产周转率 | 上升=效率改善，下降=资产堆积 | 总资产周转率(次) |
| 固定资产周转率 | 上升=产能利用率提高 | 固定资产周转率 |
| 存货周转天数 | 下降=变现加快 | 存货周转天数(天) |
| 应收账款周转天数 | 上升=放宽信用；下降=议价力弱 | 应收账款周转天数(天) |
| 上下游占款能力 | (应付+预收-应收-预付)/营收 | 越大越强势 |
| 固定资产比重 | 稳定=维持性投资 | 固定资产/总资产 |

### 上下游占款能力计算
```python
# 从资产负债表中提取：
# 应付账款、应付票据、预收款项、应付职工薪酬、应交税费等 → 经营性负债
# 应收账款、预付款项、其他应收款等 → 经营性资产
上下游占款 = (经营性负债 - 经营性资产) / 营业收入
>0 表示占用上下游资金（优势）
<0 表示被上下游占用（劣势）
```

## 模块4：现金流质量分析

### 格雷厄姆最重视的模块
> "现金流是真相，报表利润只是意见。" — 本杰明·格雷厄姆

### 核心指标
| 指标 | 计算 | 判断标准 |
|------|------|---------|
| 净现比 | 经营CF/净利润 | >1优秀，>0.8安全，<0.8警告，<0.6危险 |
| 净现比趋势 | 3年对比 | 连续2年下降=重要警告 |
| FCF充足率 | FCF/净利润 | >80%优秀 |
| 自我造血 | 经营CF vs 投资CF | 经营>投资=可持续 |
| 分红安全率 | 分红/经营CF | <70%为安全 |
| 筹资依赖度 | 筹资CF/经营CF | <20%为独立 |

### 净现比归因分析
净现比下降的可能原因：
1. **放宽信用换增长**：应收账款增速>营收增速，需检查应收周转天数
2. **原材料囤积**：采购现金流出增加，大宗商品涨价预期
3. **产能扩张期正常现象**：大量现金投入在建工程/固定资产
4. **上下游议价能力减弱**：被迫延长付款周期

判断方法：结合资产负债表（应收/存货/在建工程）和现金流量表（投资CF）综合判断

## 模块5：估值分析

### 格雷厄姆数计算（精确版）

**保守版**（格雷厄姆原教旨）：
```
Graham_Number = √(22.5 × EPS × BPS)
```
- EPS：使用加权EPS（摊薄EPS作为参考）
- BPS：使用每股净资产（调整后）

**成长调整版**（适用于ROE>15%）：
```
因子 = 1 + 0.5 × (ROE - rf)
其中 rf = 无风险利率（通常取10年国债收益率，约3%）
Graham_Number(Growth) = √(22.5 × EPS × BPS × 因子)
```

**安全边际判断**：
| 股价/格雷厄姆数 | 判断 | 操作 |
|---------------|------|------|
| < 0.7 | 深度安全边际 | 强烈买入 |
| 0.7-0.85 | 合理安全边际 | 买入 |
| 0.85-1.0 | 临界 | 持有/轻仓 |
| 1.0-1.2 | 无安全边际 | 持有不买 |
| > 1.2 | 明显溢价 | 回避 |

### 其他估值参考
| 方法 | 说明 | 适用场景 |
|------|------|---------|
| PE历史分位 | 近5年PE百分位 | 需前复权数据 |
| 股债收益率 | 1/PE vs 国债收益率 | 相对吸引力判断 |
| PEG | PE/未来3年增速 | 增速<15%时参考 |
|股息率 | 分红/股价 | 高分红股参考 |

## 模块6：护城河定性评估

### 晨星四层护城河（简化版）

1. **成本优势**（低护城河→中等护城河）
   - 规模效应
   - 地理位置优势
   - 供应链整合优势
   
2. **转换成本**（高护城河）
   - 客户切换供应商的成本和难度
   - B2B通常低于B2C
   
3. **网络效应**（非常高护城河）
   - 用户越多价值越大
   - 典型：社交平台、操作系统
   
4. **无形资产**（高护城河）
   - 品牌溢价
   - 专利/技术壁垒
   - 独家资质/牌照

### 包装印刷行业特点
- 转换成本：**低**（客户可轻易切换）
- 成本优势：**弱**（差异化小）
- 网络效应：**无**
- 无形资产：**弱**（无专利壁垒）
- **整体护城河评分**：1-3/10（低护城河行业）

## 模块8：同业比较与风险警示

### Red Flag清单（触发任一条需深入调查）
| # | 检查项 | 触发条件 |
|---|--------|---------|
| 1 | 应收增速异常 | 应收增速 > 营收增速 × 1.5 |
| 2 | 存货激增 | 存货增速 > 营收增速 × 1.5 |
| 3 | 在建工程异常 | 连续2年不转固 |
| 4 | 净现比危险 | < 0.6 |
| 5 | 大股东高质押 | > 50% |
| 6 | 毛利率持续下降 | 连续3年下降 |
| 7 | 审计师更换 | 近3年更换 |
| 8 | 非标审计意见 | 保留/无法表示 |

### 三重信号预警（框架可能失效）
当以下三重信号同时出现时，说明公司进入**低效扩张陷阱**，分析框架应提前预警：
- 总资产周转率持续下降
- 净现比持续下降
- 有息负债率上升
