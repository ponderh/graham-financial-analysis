# 格雷厄姆《证券分析》核心原则

## 格雷厄姆价值投资体系摘要

### 核心理念
1. **内在价值**：股票的价值取决于公司未来收益的资本化
2. **安全边际**：以低于内在价值的价格买入是投资的本质
3. **市场先生**：市场短期是投票机，长期是称重机
4. **盈利稳定性**：投资于有长期稳定盈利记录的公司

### 格雷厄姆的选股标准（原始版）

**防御型投资者选股7条：**
1. 公司规模适中（工业产值不低于1亿美元）
2. 财务状况稳健（流动比率>2）
3. 20年连续分红
4. 20年无亏损记录
5. EPS近10年增长超过1/3
6. PE < 15倍
7. PB < 1.5倍 或 PE×PB < 22.5

### 格雷厄姆数详解

**公式**：V = √(22.5 × EPS × BPS)

**含义**：
- EPS × BPS = 每股收益 × 每股净资产
- 22.5 = 15(合理PE上限) × 1.5(合理PB上限)
- 实际上是PE=15且PB=1.5时的估值上限

**适用条件**：
- 盈利稳定（非周期股）
- 不适用于高成长股（会严重低估）
- 成长调整版适用于ROE持续>15%的公司

### 格雷厄姆对A股的适应性调整

**需要调整的原因**：
1. A股历史短（2000年后才有完整数据）
2. 中国会计准则与美国GAAP差异
3. 地方政府补贴可能扭曲利润
4. 国企软预算约束问题
5. ESG意识薄弱

**适应调整方案**：
| 原始要求 | A股调整 |
|---------|---------|
| 7-10年盈利记录 | 至少3年年报+季度跟踪 |
| 20年连续分红 | 至少5年连续分红 |
| 流动比率>2 | 速动比率>1 + 现金比率>30% |
| PE<15 | PE<20（考虑A股溢价） |
| PB<1.5 | PB<3（考虑A股溢价） |

### 格雷厄姆最重视的财务指标（优先级排序）

1. **现金流质量**（最重要）
   - 经营现金流/净利润 > 0.8
   - 经营现金流/总资产 > 5%
   - 现金再投资比率 > 8%

2. **盈利稳定性**
   - ROE持续 > 15%
   - 非经常性损益占比 < 10%
   - 毛利率CV < 15%

3. **财务稳健性**
   - 资产负债率 < 50%（制造业<40%）
   - 有息负债率 < 30%
   - 利息保障倍数 > 5倍

4. **股东回报**
   - 分红率 30%-70%
   - 回购注销优于分红

### 格雷厄姆的风险控制原则

1. **分散投资**：至少10-30只股票，单一不超过5%
2. **止损纪律**：下跌20%重新评估
3. **定期复审**：每年重新评估持仓
4. **不预测市场**：市场波动不是卖出的理由
5. **能力圈**：只投资自己了解的行业

### 格雷厄姆的估值局限性

**不适用于**：
- 周期性行业（钢铁、煤炭、航空等）
- 金融行业（银行、保险有特殊估值方法）
- 亏损公司
- 高成长股（格雷厄姆数会严重低估）
- 重组/并购类公司

**适用**：
- 稳定增长的消费股
- 公用事业
- 成熟期制造业
- 寡头垄断行业
