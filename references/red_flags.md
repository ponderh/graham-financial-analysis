# Red Flag风险警示清单

## 8条必须检查的Red Flags

| # | 检查项 | 触发条件 | 严重程度 | 应对措施 |
|---|--------|---------|---------|---------|
| 1 | 应收增速异常 | 应收增速 > 营收增速 × 1.5 | 🔴 高 | 深入调查信用政策变化，计算账龄结构 |
| 2 | 存货激增 | 存货增速 > 营收增速 × 1.5 | 🔴 高 | 检查存货结构（原材料/在产品/产成品），判断滞销风险 |
| 3 | 在建工程异常 | 连续2年不转固 | 🟡 中 | 在建工程利息费用化可能美化利润，检查转固时间表 |
| 4 | 净现比危险 | < 0.6 | 🔴 高 | 净利润含金量存疑，可能存在利润操纵 |
| 5 | 大股东高质押 | > 50% | 🔴 高 | 代理风险极高，大股东可能掏空上市公司 |
| 6 | 毛利率持续下降 | 连续3年下降 | 🟡 中 | 竞争加剧或定价权丧失，行业可能进入衰退期 |
| 7 | 审计师更换 | 近3年更换 | 🟡 中 | 可能存在财务问题，需深入调查更换原因 |
| 8 | 非标审计意见 | 保留/无法表示 | 🔴 极高 | 直接排除，不适合格雷厄姆框架 |

## 进阶Red Flags（需深入分析）

### 利润质量类
- **收入确认异常**：季度收入分布异常均匀（可能虚构）
- **应收账款账龄恶化**：1年以上应收占比增加
- **存货周转放慢**：但毛利率未下降（可能隐藏成本）
- **预付款异常增加**：可能虚增利润或关联交易
- **其他应收款持续高位**：可能隐藏资金占用

### 现金流类
- **经营现金流与净利润长期背离**：持续>3年需警惕
- **投资现金流大额流出但无营收增长**：可能资金去向不明
- **筹资现金流依赖度高**：>经营现金流30%说明公司不独立
- **商誉剧增后大额减值**：并购泡沫破裂
- **在建工程长期不转固**：利息费用化或工程烂尾

### 资产负债表类
- **货币资金与有息负债双高**：可能资金受限
- **其他应付款异常**：可能隐藏负债
- **固定资产原值与产能不匹配**：可能虚增资产
- **无形资产剧增**：可能虚增估值
- **少数股东权益异常**：可能稀释上市公司利益

### 公司治理类
- **高股权质押**：>50%极度危险
- **频繁关联交易**：金额>营收10%需警惕
- **管理层频繁变动**：尤其是CFO/审计负责人
- **实际控制人变更**：战略方向可能改变
- **诉讼仲裁大量累积**：或有负债风险

## 三重信号预警系统

当以下**三重信号同时出现**，说明公司进入**低效扩张陷阱**，框架应提前预警：

1. **总资产周转率持续下降**（资产效率降低）
2. **净现比持续下降**（盈利含金量下降）
3. **有息负债率上升**（财务杠杆提高）

**触发三重信号后**：
- 立即停止加仓
- 深入调查38亿/大额资本支出去向
- 评估管理层资本配置能力
- 考虑减仓或清仓

## 格雷厄姆框架失效条件

| 条件 | 说明 |
|------|------|
| 行业周期底部 | 格雷厄姆数会低估周期股价值 |
| 高成长阶段 | 格雷厄姆数会严重低估 |
| 会计准则变更 | 跨期比较失真 |
| 重大资产重组 | 历史数据不可比 |
| 新业务转型 | 原有护城河消失 |
| 管理层变更 | 战略方向不确定 |

## A股特有Red Flags

| 检查项 | 说明 |
|--------|------|
| 地方政府补助 | 可能扭曲主业盈利真实性 |
| 税收优惠 | 优惠到期后盈利断崖风险 |
| 出口退税 | 汇率政策影响大 |
| 商誉减值 | 并购后遗症 |
| 股权激励摊薄 | EPS被稀释 |
| 大小非解禁 | 减持压力 |
| 财报披露滞后 | 年报与季报时间差 |
